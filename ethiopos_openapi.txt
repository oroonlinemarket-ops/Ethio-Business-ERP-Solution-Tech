    ## EthioPOS — 21-Module ERP API for Ethiopian & African SMEs
    
    Full REST API covering POS, Inventory, Accounting, HR, Invoices,
    Warehouse, CRM, AI Intelligence, and more.
    
    ### Authentication
    All endpoints (except `/auth/*` and payment webhooks) require a
    JWT Bearer token obtained from `POST /auth/login`.
    
    ### Ethiopian Compliance
    - Currency: **ETB (Ethiopian Birr)**
    - VAT: **15%** per ERCA regulations
    - Payroll: Ethiopian Labour Proclamation No. 377/2003
    - Pension: Employee **7%**, Employer **11%**
    
    ### Multi-tenancy
    Every request is scoped to the authenticated user's `tenant_id`.
    No cross-tenant data leakage is possible.
